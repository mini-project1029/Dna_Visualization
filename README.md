# Dna_Visualization

## Description
A system to visualize DNA in 3D and detect mutations and motifs.

## Features
- 3D DNA Helix
- Mutation Highlighting
- Motif Detection
- GC Content Analysis
- Interactive UI

## Tech Stack
Frontend: React, Three.js  
Backend: (Python)  
ML: Scikit-learn  

## How to Run

Backend:
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

Frontend:
cd frontend
npm install
npm run dev
